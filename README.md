KCMS 2.0
===============

> 运行环境要求PHP7.1+  
> 基于ThinkPHP 6.x

## 主要功能

* 开箱即用的cms内容管理系统
* 可作为基本的后台系统的基础框架
* 封装常用的Model方法
* 采用传统的mvc模式
* 权限管理
* 代码生成器

## 安装

下载:
~~~
git clone https://gitee.com/kitty_18php/kcms_v2.git kcms
~~~

安装:
~~~
composer install
~~~
